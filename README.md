La serie de Fibonacci se comienza con el 0 y 1, susecivamente se sumara el antecesor y el sucesor.

0 + 1 = 1

que dando asi 0,1,1

1 + 1 = 2

0,1,1,2

1 + 2 = 3

0,1,1,2,3

0,1,1,2,3,5

3 + 5 = 8

5 + 8 = 11

0,1,1,2,3,5,8

5 + 8 = 13
